# Share-Plane-Crazy-Builds

Share-Plane-Crazy-Builds is a Lua script designed for game executors that enables importing and exporting builds using JSON format.

This tool allows users to easily share, save, and restore custom builds in a structured and portable way. The JSON-based system makes builds readable, editable, and transferable between sessions or users.

## Features

* Import builds from JSON files
* Export builds to JSON format
* Simple and lightweight implementation
* Open to community improvements
